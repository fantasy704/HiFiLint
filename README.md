# HiFiLint

HiFiLint 是一款轻量级的 Android 音频诊断工具，旨在帮助音频发烧友和开发者直观地查看系统真实的音频输出参数，精准识别 **音频降采样 (Downsampling)** 和 **SRC (采样率转换)** 问题。

## 🌟 核心功能

*   **音频参数实时查看**：显示当前播放音频的采样率、位深及声道信息。
*   **降采样检测**：识别系统是否将高规格音频（如 96kHz/192kHz）强制重采样至 48kHz 或 44.1kHz 输出。
*   **通路诊断**：确认音频真实的输出设备（扬声器、蓝牙、USB DAC 等）及物理地址。
*   **高级模式 (Shizuku)**：可选功能。通过 Shizuku 获取底层 `dumpsys media.audio_flinger` 原始数据，提供最权威的系统级诊断报告。

在 Android 平台上，即便音乐 App 提示正在播放 “Hi-Res” 音频，系统底层仍可能因为驱动或策略原因进行重采样（SRC）。HiFiLint 通过读取安卓音频引擎的运行状态，让你看清每一帧音频在交给硬件前的真实模样。

## 🛠️ 使用方式

1.  **普通模式**：直接打开应用，查看基础的音频输出信息。
2.  **高级模式 (推荐)**：
    *   安装并激活 [Shizuku](https://shizuku.rikka.app/)。
    *   在 HiFiLint 中授权 Shizuku 访问。
